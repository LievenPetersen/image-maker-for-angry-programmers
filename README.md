# primitive pixel art tool
- to build: `make` or `cc main.c -lraylib -lm`
- to build & run: `make run`


- Image is always saved as `out.png`

- raylib + raygui
- requires raylib installation (4 or 5 should be fine) (`sudo make install` in raylib/src)
